About This Guide

All website content — text, images, prices, contact details, services — is managed through the Admin Panel at /admin/. You do not need to edit any code files to update the website.

1 — Getting Started & Admin Access

Accessing the Admin Panel

The Admin Panel is where all website content is managed. It is only accessible to authorised users.

1

Open your browser and go to: http://127.0.0.1:8000/admin/ (local) or https://yourdomain.com/admin/ (live site)

2

Enter your Username and Password

3

Click Log In — you will see the Management Dashboard

Creating a New Admin User

Run this command in the terminal from the project folder:

python manage.py createsuperuser

You will be prompted to enter a username, email address, and password.

Resetting a Forgotten Password

If you forgot your admin password:

Run: python manage.py changepassword <your_username>

To see all admin users: python manage.py shell -c "from django.contrib.auth.models import User; print([u.username for u in User.objects.filter(is_superuser=True)])"

4 — Freight Cost Calculator

The trucking cost calculator on the homepage is entirely admin-controlled. No code changes are needed to update rates, add vessel sizes, or manage cargo types.

4.1 — Vessel Sizes (Truck Types)

Controls the truck type dropdown in the calculator and the per-mile rate used in cost calculations.

Admin → Vessel Sizes → Add Vessel Size / Click to edit
FieldDescription
Name REQUIREDDisplayed in the dropdown (e.g. "Dry Van 53'")
Key REQUIREDUnique identifier — lowercase, hyphens only (e.g. dry-van). Used internally.
DescriptionShort description shown to users
CapacityLoad capacity info (e.g. "44,000 lbs / 2,700 cu ft")
Rate Per Mile REQUIREDBase cost in USD per mile (e.g. 2.80). This is the primary pricing variable.
Min Charge REQUIREDMinimum job cost in USD regardless of distance (e.g. 600)
Fuel Surcharge %Fuel surcharge percentage added on top of base rate (e.g. 25 = 25%). Default: 25.
Is ActiveUncheck to hide this option from the calculator without deleting it
OrderDisplay order in the dropdown
How the cost formula works:

Base Rate = max(Min Charge, Rate Per Mile × Distance Miles)
Fuel Surcharge = Base Rate × Fuel %
Cargo Adjustment = Base Rate × (Cargo Multiplier − 1)
Total = Base Rate + Fuel Surcharge + Cargo Adjustment (shown as ±10% range)

4.2 — Cargo Types

Controls the material/cargo type dropdown and applies a multiplier to the base rate.

Admin → Cargo Types → Add Cargo Type / Click to edit
FieldDescription
Name REQUIREDDisplayed in the dropdown (e.g. "Hazardous Materials")
Key REQUIREDUnique identifier (e.g. hazmat)
DescriptionBrief description of this cargo type
Multiplier REQUIREDCost multiplier. 1.00 = no surcharge. 1.50 = 50% surcharge. 2.00 = double the base rate.
Is ActiveUncheck to hide from the calculator
OrderDisplay order in the dropdown
Multiplier examples:

General Freight = 1.00 (no change)  |  Hazmat = 1.50 (+50%)  |  Oversized = 1.80 (+80%)

4.3 — Port Cities

The list of cities available in the calculator's origin/destination search. Users type a city name and select from this list.

Admin → Port Cities → Add Port / City / Click to edit
FieldDescription
Name REQUIREDCity name as it appears in the search (e.g. "Los Angeles")
Country REQUIREDCountry name (e.g. "USA", "Canada", "UK"). Shown alongside the city name.
ZoneGeographic zone used for routing. Select the nearest zone from the list.
LatLatitude coordinate (e.g. 34.052). Required for distance calculation.
LngLongitude coordinate (e.g. -118.244). Required for distance calculation. Negative = West.
Is ActiveUncheck to hide from the calculator without deleting
Finding coordinates:

Go to maps.google.com, right-click on any city, and click the coordinates shown at the top of the menu. Copy the two numbers — the first is latitude, the second is longitude.

13 — Running the Website Locally

Use these commands to start the website on your computer for development and testing.

Starting the Server

1

Open Terminal (Mac) or Command Prompt (Windows)

2

Navigate to the project folder:
cd /Users/supriyayadav/seaways-django

3

Activate the virtual environment:
source venv/bin/activate

4

Start the server:
python manage.py runserver

5

Open browser and go to: http://127.0.0.1:8000/

Port already in use?

If you see "Error: That port is already in use", run:
kill -9 $(lsof -t -i:8000)   then   python manage.py runserver

Common Commands Reference

CommandWhat It Does
python manage.py runserverStarts the development web server
python manage.py seedResets and repopulates the database with default data Warning: overwrites existing data
python manage.py createsuperuserCreates a new admin user
python manage.py changepassword <username>Resets an admin user's password
python manage.py makemigrationsPrepares database schema changes (for developers)
python manage.py migrateApplies database schema changes (for developers)
⚠ Warning about python manage.py seed:

Running the seed command will delete and recreate all services, blog posts, team members, companies, and jobs with the default data. It will NOT delete quotes, contact messages, or custom data you have entered through the admin. Only run this to reset demo content.
